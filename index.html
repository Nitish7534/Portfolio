Projects


Whatsapp Chat Analyzer:


This project focuses on analyzing WhatsApp chat data to extract meaningful insights. Using Python, Pandas, and Regular Expressions (Regex), the raw chat text is processed to structure messages by date, time, user, and message content. Key features include message frequency analysis, active user identification, response time calculation, and time-based messaging patterns. Additionally, Streamlit is used to create an interactive dashboard for data visualization, making it easy to explore trends such as peak chatting hours, most active participants, and daily messaging behavior. This project demonstrates data processing, natural language handling, and visualization skills, making it valuable for social media analysis and business communication insights.

Paws And Purpose:


Paws and Purpose is a WordPress-based website designed to support animal welfare initiatives. Built using the Divi theme, the platform provides an interactive and user-friendly experience to encourage donations, spread awareness, and facilitate engagement. The website features a live chat integration that enables real-time communication for queries and support, ensuring seamless interaction between users and the organization. To streamline the donation process, a secure payment gateway has been integrated, allowing users to contribute effortlessly to the cause. Additionally, a dedicated FAQs section provides answers to common concerns, educating visitors about animal welfare and the organization's mission. The website also includes submittable forms, enabling users to report animals in distress, sign up as volunteers, or share their personal experiences related to animal welfare. This project reflects my expertise in WordPress development, UI/UX customization, and functional integrations, demonstrating my ability to create a meaningful and impactful digital presence for a social cause. TravelZoo:


TravelZoo is a dynamic trip planner website designed to enhance the travel experience by providing essential information and seamless booking functionality. Developed using HTML, CSS, and JavaScript, the platform offers an intuitive and visually appealing interface for users to plan their trips efficiently. One of the key features of TravelZoo is the integration of a Weather API, which allows users to check the weather conditions of their desired destination before visiting, ensuring better travel planning. Additionally, the website includes a submittable booking form, enabling users to make reservations conveniently. This project showcases my proficiency in front-end web development, API integration, and interactive UI design, demonstrating my ability to build functional and user-centric web applications.
